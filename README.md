# RESERVED
**Description:** This is a venue reservation mobile application. This application has two sides, one for admins and one for normal users.

## Technologies Used
- **React Native**
- **TypeScript**
- **Expo**
- **REST API** (REST API is no longer active)
- **Image Picker**
- **Redux Toolkit**
